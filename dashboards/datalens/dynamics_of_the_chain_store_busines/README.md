## Дашборд, демонстрирующий основные метрики сети магазинов
* Динамика бизнеса: динамика количества заказов, динамика выручки, динамика чека
* Рейтинг магазинов по выручке
* Таблица с продажами по территориям

## Данные, на основе которых был создан дашборд (csv-файлы, SQL-script) находятся в папке **[data](https://github.com/neonobot/portfolio/tree/main/dashboards/datalens/dynamics_of_the_chain_store_busines/data)**
